# Microve.ai Site Structure

This folder contains the simplified static site.

## Pages

- `index.html` - Homepage (high-level message + differentiation)
- `what-we-do.html` - Detailed explanation + automation type comparison
- `calculator.html` - Hours/value calculator
- `contact.html` - Simple contact form

## Notes

- This is a lightweight static build (no framework required).
- Navigation links are all relative across these four pages.
